drupal-geany-widget
===================

A Drupal snippets that provide auto-complete functionality for Geany.

INSTALLATION
------------

Copy these `.php.tags` files and put it in `~.config/geany/tags` directory.

SUPPORTS
--------

1. Drupal core
2. Views
3. Date
4. Smart trim
5. Ctools
6. Oembed
7. Entity
8. Nodequeue
9. Pathauto
10. Token
11. Panels
12. Workbench
13. Libraries
14. Diff
15. Context
16. Metatag
