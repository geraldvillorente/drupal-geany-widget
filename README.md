drupal-geany-widget
===================

A Drupal snippets that provide auto-complete functionality for Geany.

INSTALLATION
------------

Copy these two `.php.tags` files and put it in `~.config/geany/tags` directory.

SUPPORTS
--------

1. Drupal core
2. Views
3. Date
4. Smart trim
5. Elysia cron
